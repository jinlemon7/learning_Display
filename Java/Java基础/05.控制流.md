# 条件表达式的本质是布尔值 true/false
  Java不允许将数字当作布尔值使用  

# 逗号
  - 逗号分隔符：分隔声明/函数参数
  - 逗号操作符：for循环中定义多个同类变量，多个步进语句  
    例：for(int i = 1, j = i + 10; i < 5; i++, j = i * 2)

# Java标签与break/continue
  标签：唯一的放置位置是迭代语句之前，中间没有任何语句  

  Java中使用标签的唯一场景：用到了嵌套循环，且需要使用break/continue跳出多层的循环  
  例：  
  ```java
  outer:
  while(true){
    for( ; i<10; i++){
      if(i==0){i++; continue outer;}
      if(i==1){break outer;}
    }
  }
  ```
  continue outer / break outer均会直接退出两层循环，不执行for循环的步进操作。  
  区别在于continue会再次进入outer后的循环，break会跳过outer后的循环，执行后面的代码。  

# switch
  如果一个case末尾没有break，后续的case均会执行，直到遇到一个break。  
  switch选择器（case后选项）可以为：  
  - int
  - char
  - String

# 构造器
  在类中与类名相同的方法
  
# 方法重载
  方法名相同，但以不同的参数列表区分。例：重载构造器
  
# this关键字（只能在非static方法中使用）
  用途：将当前对象传递给另一个对象/类的方法（外部方法）

# 构造器中调用构造器
  1. 不能同时使用this调用两个另外的构造器
  2. 构造器调用必须出现在构造器方法最开始的部分，否则编译器报错
  3. 若方法参数与成员属性重名，则用this.属性名表示类成员数据 例：this.s = s
  4. 非构造器方法不允许调用构造器
  ```java
  类名(...){
    this('xxx');     //this()自动匹配参数列表匹配的构造器
    ...
  }
  ```

# 排序和搜索

## 排序sort(),默认升序
  `Arrays.sort(array);`    //对数组排序  
  `Collections.sort(list);`    //对容器排序  

  - 逆序排序：使用Collection.reverseOrder()  
    `Arrays.sort(array, Collection.reverseOrder() );`  
    `Collection.sort(list, Collection.reverseOrder() );`  
  - 自定义逆序排序：使用Comparator.reversed()  
    `Comparator<T> by = (o1, o2) -> o1 - o2;`  
    `Collections.sort(list, by.reversed() );`
    
  - 方法引用排序  
    - 静态方法：`ClassName.sort(instanceName::FunctionName);`
    - 实例方法：`Collections.sort(className, Comparator.comparing(instanceName::FunctionName) );`

  - 多级排序  
    `Comparator<T> by = Comparator.comparing(T:: F1Name).thenComparing(T:: F2Name);`  
    先按F1Name排序，再按F2Name排序  
    Comparator.comparing(F1Name)将Function指定为重写的compare()

  - 并行排序：利用多核处理器分治-组合排序  
    ```java
    Arrays.parallelSort(array);
    list.parallelStream().sorted().collect(Collections.toList() );
    ```
  
### 自定义排序
  自定义Comparator  
  ```java
  Arrays.sort(array, new Comparator<T>(){         //匿名泛型类
    @Override
    public int compare(T o1, T 02){
      return o1 - o2;                            //此处决定用来排序的对象（可为类的字段）和排序顺序
      //o1.compareTo(o2);                        //参数1 - 参数2 或 参数1.compareTo(参数2)：升序
    }                                            //参数2 - 参数1 或 参数2.compareTo(参数1)：降序
  };
  ```

  Comparator可使用Lambda表达式简化  
  ```java
  Comparator<T> by =  (T o1, T o2) -> o1 - o2;    //Lambda表达式相当于重写compare()
  Arrays.sort(array, by);
  ```

# 字符串处理

## 大小写转换
  Character类转换                //支持所有Unicode字符  
  Character.toUpperCase(c);     //安全可靠，自动处理非字母字符（数字，符号不变化）  
  Character.toLowerCase(c);     //c为字符/字符串  
  
## 判断char类型字符的大小写
  Character.isUpperCase(c);  
  Character.isLowerCase(c);

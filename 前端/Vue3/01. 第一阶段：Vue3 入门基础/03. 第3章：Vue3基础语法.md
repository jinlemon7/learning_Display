## 直接在页面中使用 Vue
  对于学习和简单‌‌‌原型开发，你也可以直接通过 ⁢⁢⁢CDN 在 HTML 页面中‏‏‏使用 Vue3：    
  ```bash
  <!DOCTYPE html>
  <html lang="zh-CN">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的第一个 Vue 应用</title>
    <!-- 引入 Vue3 -->
    <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
  </head>
  <body>
    <!-- Vue 应用将挂载到这个 div -->
    <div id="app"></div>
    <script>
      // 使用 Vue3 的组合式 API
      const { createApp, ref } = Vue
      
      createApp({
        setup() {

  ```

  在这个简单的例子中，我们使用了 Vue3 的核心特性——组合式 API。setup() 函数是组合式 API 的入口点，而 ref() 是创建响应式引用的核心函数。  

# 组合式 API 基础
  Vue3 的核心特性是‌‌‌组合式 API（Composition A⁢⁢⁢PI），它提供了更灵活的代码组织方式和逻辑‏‏复用‏能力。相比于 Vue2 的选项式 AP‍‍‍I，组合式 API 允许我们按照逻辑关注点⁡⁡⁡组织代码，而不是按照选项类型。  

## setup 函数
  setup 函数是组合式 API 的核心，它在组件实例被创建之前执行，因此在 setup 中没有 this。setup 函数接收 props 和 context 作为参数：  

  ```javascript
  import { ref } from 'vue'
  
  export default {
    props: {
      title: String
    },
    setup(props, context) {
      console.log(props.title) // 访问传入的 props
      
      // 创建响应式状态
      const count = ref(0)
      
      // 定义函数
      function increment() {
        count.value++
      }
      
      // 返回需要暴露给模板的内容
      return {
        count,
        increment
      }
    }
  }
  ```

## script setup 语法糖
  在 Vue3 的单文件组件中，我们可以使用 <script setup> 语法糖来简化组合式 API 的使用：  
  ```html
  <script setup>
  import { ref } from 'vue'
  
  // 响应式状态
  const count = ref(0)
  
  // 方法
  function increment() {
    count.value++
  }
  </script>
  <template>
    <button @click="increment">计数: {{ count }}</button>
  </template>
  ```
  使用 <script setup> 有以下优势：  
  - 更少的样板代码
  - 自动暴露变量和导入的组件
  - 更好的类型推导
  - 更好的运行时性能

## 响应式 API
  Vue3 提供了两个核心响应式 API：ref 和 reactive  

### ref
  ref 用于创建任意类型值的响应式引用：   
  ```javascript
  import { ref } from 'vue'

  // 创建一个初始值为 0 的响应式引用
  const count = ref(0)
  
  // 在 JavaScript 中需要通过 .value 访问或修改值
  console.log(count.value) // 0
  
  // 修改值会触发视图更新
  count.value++
  ```

  需要注意的是，在模板中使用 ref 时，会自动解包，不需要写 .value：  
  ```html
  <template>
    <!-- 不需要 .value -->
    <p>计数: {{ count }}</p>
  </template>
  ```

### reactive
  reactive 用于创建对象的响应式代理：  
  ```javascript
  import { reactive } from 'vue'
  
  // 创建响应式对象
  const state = reactive({
    count: 0,
    name: '张三'
  })
  
  // 直接访问或修改属性
  console.log(state.count) // 0
  state.count++
  ```

  与 ref 不同，reactive 不需要使用 .value，但它只能用于对象类型（包括数组和 Map、Set 等）。  

## 计算属性-computed()
  计算属性是‌‌‌基于响应式状态⁢派⁢生⁢出的值，它‏会缓‏存结‏果并在‍依赖变‍化时自‍动⁡更新：  
  ```javascript
  import { ref, computed } from 'vue'
  
  const firstName = ref('张')
  const lastName = ref('三')
  
  // 创建计算属性
  const fullName = computed(() => {
    return firstName.value + lastName.value
  })
  
  console.log(fullName.value) // '张三'
  firstName.value = '李'
  console.log(fullName.value) // '李三'
  ```

  在 <script setup> 中使用：  
  ```html
  <script setup>
  import { ref, computed } from 'vue'
  
  const firstName = ref('张')
  const lastName = ref('三')
  const fullName = computed(() => firstName.value + lastName.value)
  </script>
  <template>
    <p>全名: {{ fullName }}</p>
  </template>
  ```

## 侦听器
  当需要在状‌‌‌态变化时执行副作用⁢操作⁢⁢（如异步请求）‏时，可以‏‏使用侦听器‍：    
  ```javascript
  import { ref, watch } from 'vue'
  
  const searchQuery = ref('')
  const searchResults = ref([])
  const isLoading = ref(false)
  
  // 侦听 searchQuery 的变化
  watch(searchQuery, async (newQuery, oldQuery) => {
    if (newQuery.trim()) {
      isLoading.value = true
      try {
        const results = await fetchSearchResults(newQuery)
        searchResults.value = results
      } finally {
        isLoading.value = false
      }
    } else {
      searchResults.value = []
    }
  })
  ```

  Vue3 还提供了 watchEffect，它会自动跟踪回调函数中使用的响应式状态：  
  ```javascript
  import { ref, watchEffect } from 'vue'
  
  const id = ref(1)
  const data = ref(null)
  
  watchEffect(async () => {
    // id.value 被自动跟踪
    data.value = await fetchData(id.value)
  })
  ```

## 生命周期钩子
  Vue3 的组合式 API 提供了一组生命周期钩子函数，可以在 setup 中使用：    
  ```javascript
  import { onMounted, onUpdated, onUnmounted } from 'vue'
  
  export default {
    setup() {
      onMounted(() => {
        console.log('组件已挂载')
      })
      
      onUpdated(() => {
        console.log('组件已更新')
      })
      
      onUnmounted(() => {
        console.log('组件已卸载')
      })
    }
  }
  ```

  在 <script setup> 中：  
  ```html
  <script setup>
  import { onMounted } from 'vue'
  
  onMounted(() => {
    console.log('组件已挂载')
  })
  </script>
  ```

# 模板语法基础
  Vue 使‌‌‌用基于 HTML⁢⁢ ⁢的模板语法，使‏‏你能‏够声明式地将‍‍渲染后‍的 DOM⁡⁡ 绑定到⁡组件实例的数据。  

## 文本插值
  最基本的数据绑定形式是文本插值，使用"双大括号"语法：  
  ``` html
  <span>消息: {{ message }}</span>
  ```

  双大括号会被替换为相应组件实例中 message 属性的值。当这个属性变化时，视图会自动更新。  

## 原始 HTML
  如果你想渲染 HTML 而非纯文本，可以使用 v-html 指令：  
  ```html
  <p>使用插值: {{ rawHtml }}</p>
  <p>使用指令: <span v-html="rawHtml"></span></p>
  ```

  ```javascript
  const rawHtml = ref('<span style="color: red">这是红色文本</span>')
  ```

  安全提示：动态渲染 HTML 容易导致 XSS 漏洞，只对可信内容使用 v-html，永远不要用于用户提供的内容。  

## 属性绑定
  使用 v-bind 指令（简写为 :）绑定 HTML 属性：  
  ```html
  <div v-bind:id="dynamicId"></div>
  <!-- 简写 -->
  <div :id="dynamicId"></div>
  ```

  如果绑定的值是 null 或 undefined，则该属性将被移除。  

  布尔属性的‌‌‌特殊处理：  ⁢   
  ```html
  <button :disabled="isButtonDisabled">按钮</button>
  ```

  如果 isButtonDisabled 是真值或空字符串，按钮会被禁用。  

## 使用 JavaScript 表达式
  Vue 支‌‌‌持在数据绑定中⁢使⁢用⁢完整的 J‏av‏aS‏cri‍pt ‍表达式‍：  
  ```html
  {{ number + 1 }}
  
  {{ ok ? '是' : '否' }}
  
  {{ message.split('').reverse().join('') }}
  
  <div :id="`list-${id}`"></div>
  ```

  需要注意的‌‌‌是，每个绑定只⁢能⁢包⁢含单个表达‏式，‏而不‏能是语‍句或‍流控制。  

# 指令
  指令是带有 v- 前缀的特殊 attribute，用于对 DOM 执行响应式操作。  

## 常用指令
### v-if 和 v-show：条件性渲染元素
  ```html
  <p v-if="seen">现在你看到我了</p>
  <p v-show="seen">我通过 CSS display 属性控制显示</p>
  ```

### v-for：基于数组或对象渲染列表
  ```html
  <ul>
    <li v-for="item in items" :key="item.id">
      {{ item.text }}
    </li>
  </ul>
  ```

### v-on：绑定事件监听器（简写为 @）
  ```html
  <button v-on:click="increment">增加</button>
  <!-- 简写 -->
  <button @click="increment">增加</button>
  ```

### v-model：创建双向数据绑定，主要用于表单元素
  ```html
  <input v-model="message">
  ```

### v-bind：动态绑定 HTML 属性（简写为 :）
  ```html
  <img v-bind:src="imageSrc">
  <!-- 简写 -->
  <img :src="imageSrc">
  ```

### 指令修饰符
  指令可以带有修饰符，用于指定特殊的绑定行为：  
  ```html
  <!-- 阻止默认事件 -->
  <form @submit.prevent="onSubmit"></form>
  <!-- 按键修饰符 -->
  <input @keyup.enter="submit">
  
  <!-- v-model 修饰符 -->
  <input v-model.trim="message">
  ```

# Class 与 Style 绑定
  操作元素的 class 列表和内联样式是数据绑定的常见需求，Vue 提供了增强的 v-bind 功能来处理这两种情况。  

## 绑定 HTML Class
  对象语法：‌         
  ``` html
  <script setup>
  import { ref } from 'vue'
  
  const isActive = ref(true)
  const hasError = ref(false)
  </script>
  <template>
    <div :class="{ active: isActive, 'text-danger': hasError }"></div>
  </template>
  ```

  计算属性：‌‌      
  ```html
  <script setup>
  import { ref, computed } from 'vue'
  
  const isActive = ref(true)
  const error = ref(null)
  
  const classObject = computed(() => ({
    active: isActive.value,
    'text-danger': error.value,
    'is-disabled': !isActive.value
  }))
  </script>
  <template>
    <div :class="classObject"></div>
  </template>
  ```

  数组语法：‌‌    
  ```html
  <script setup>
  import { ref } from 'vue'
  
  const activeClass = ref('active')
  const errorClass = ref('text-danger')
  const isActive = ref(true)
  </script>
  <template>
    <div :class="[activeClass, errorClass]"></div>
    <!-- 条件应用 -->
    <div :class="[isActive ? activeClass : '', errorClass]"></div>
    <!-- 数组中使用对象 -->
    <div :class="[{ active: isActive }, errorClass]"></div>
  </template>
  ```

## 绑定内联样式
  对象语法：‌  
  ```html
  <script setup>
  import { ref } from 'vue'
  
  const activeColor = ref('red')
  const fontSize = ref(16)
  </script>
  <template>
    <div :style="{ color: activeColor, fontSize: fontSize + 'px' }"></div>
  </template>
  ```

## 直接绑定样式对象：
  ```html
  <script setup>
  import { reactive } from 'vue'
  
  const styleObject = reactive({
    color: 'red',
    fontSize: '16px',
    fontWeight: 'bold'
  })
  </script>
  <template>
    <div :style="styleObject"></div>
  </template>
  ```

# 实用示例：计数器应用
  下面是一个‌‌‌使用 Vue3⁢ ⁢组⁢合式 AP‏I ‏的计‏数器应‍用示例‍，它结‍合⁡了我们学⁡习的多个⁡概念：  
  ```vue

  import { ref, computed } from 'vue'
  
  // 响应式状态
  const count = ref(0)
  
  // 计算属性
  const countClass = computed(() => {
    if (count.value > 0) return 'positive'
    if (count.value < 0) return 'negative'
    return ''
  })

    if (count.value > 0) return '正数'
    if (count.value < 0) return '负数'
    return '零'
  })
  
  // 方法
  function increment() {
    count.value++
  }
  
  function decrement() {
    count.value--
  }
  
  function reset() {
    count.value = 0
  }
  </script>
  <template>
    <div class="counter">
      <h2>Vue3 计数器</h2>
      
      <div class="display">
        <div class="count" :class="countClass">{{ count }}</div>
        <p>当前是: {{ countDescription }}</p>
      </div>
      
      <div class="buttons">
        <button @click="decrement">减少</button>
        <button @click="reset">重置</button>
  ```

## 练习 1：创建一个待办事项组件
  创建一个简‌‌‌单的待办事项（T⁢o⁢do）⁢组件，可‏以添‏加、标记完成‏‍和删除待‍办事项。  

  答案：  
  ```vue
  <script setup>
  import { ref } from 'vue'
  
  // 响应式状态
  const newTodo = ref('')
  const todos = ref([
    { id: 1, text: '学习 Vue3 基础', done: true },
    { id: 2, text: '创建一个项目', done: false },
    { id: 3, text: '学习组合式 API', done: false }
  ])
  let nextId = 4
  
  // 方法
  function addTodo() {
    if (newTodo.value.trim()) {
      todos.value.push({
        id: nextId++,
        text: newTodo.value,
        done: false
      })
      newTodo.value = ''
    }
  }
  
  function toggleTodo(todo) {
    todo.done = !todo.done
  }
  
  function removeTodo(id) {
    todos.value = todos.value.filter(todo => todo.id !== id)
  }
  </script>
  <template>
    <div class="todo-app">
      <h2>待办事项</h2>
      
      <div class="add-todo">
        <input 
          v-model="newTodo" 
          placeholder="添加新的待办事项" 
          @keyup.enter="addTodo"
        >
        <button @click="addTodo">添加</button>
      </div>
      
      <ul class="todo-list">
        <li 
          v-for="todo in todos" 
          :key="todo.id"
          :class="{ done: todo.done }"
        >
          <input 
            type="checkbox"
            :checked="todo.done"
            @change="toggleTodo(todo)"
          >
          <span class="todo-text">{{ todo.text }}</span>
          <button @click="removeTodo(todo.id)" class="delete-btn">删除</button>
        </li>
      </ul>
      
      <p v-if="todos.length === 0">
        没有待办事项，请添加一些！
      </p>
    </div>
  </template>
  <style scoped>
  .todo-app {
    max-width: 400px;
    margin: 0 auto;
    padding: 20px;
    font-family: sans-serif;
  }
  
  .add-todo {
    display: flex;
    margin-bottom: 20px;
  }
  ```

## 练习 2：创建一个颜色选择器组件
  创建一个简‌‌‌单的颜色选择器⁢，⁢可⁢以调整 R‏GB‏ 值‏并实时预‍‍览颜色。  
  答案：  
  ```vue
  <script setup>
  import { ref, computed } from 'vue'
  
  // 响应式状态
  const red = ref(128)
  const green = ref(128)
  const blue = ref(128)
  
  // 计算属性
  const hexColor = computed(() => {
    const r = red.value.toString(16).padStart(2, '0')
    const g = green.value.toString(16).padStart(2, '0')
    const b = blue.value.toString(16).padStart(2, '0')
    return `#${r}${g}${b}`
  })
  
  const rgbColor = computed(() => {
    return `rgb(${red.value}, ${green.value}, ${blue.value})`
  })
  
  // 方法
  function randomColor() {
    red.value = Math.floor(Math.random() * 256)
    green.value = Math.floor(Math.random() * 256)
    blue.value = Math.floor(Math.random() * 256)
  }
  </script>
  <template>
    <div class="color-picker">
      <h2>颜色选择器</h2>
      
      <div class="color-preview" :style="{ backgroundColor: rgbColor }"></div>
      
      <div class="color-values">
        <p>HEX: {{ hexColor }}</p>
        <p>RGB: {{ rgbColor }}</p>
      </div>
      
      <div class="color-sliders">
        <div class="slider-group">
          <label>红 (R):</label>
          <input 
            type="range" 
            v-model.number="red" 
            min="0" 
            max="255"
          >
          <span>{{ red }}</span>
        </div>
        
        <div class="slider-group">
          <label>绿 (G):</label>
          <input 
            type="range" 
            v-model.number="green" 
            min="0" 
            max="255"
          >
          <span>{{ green }}</span>
        </div>
        
        <div class="slider-group">
          <label>蓝 (B):</label>
          <input 
            type="range" 
            v-model.number="blue" 
            min="0" 
            max="255"
          >
          <span>{{ blue }}</span>
  ```

# 计算属性与侦听器
## 计算属性 (computed)
  计算属性是‌‌‌基于响应式依赖进行⁢⁢⁢缓存的计算结果。只‏‏‏有当依赖的响应式状‍‍‍态发生变化时，计算⁡⁡⁡属性才会重新计算。  

### 基本用法
  ```javascript
  import { ref, computed } from 'vue'
  
  const firstName = ref('张')
  const lastName = ref('三')
  
  // 创建一个计算属性
  const fullName = computed(() => {
    return firstName.value + lastName.value
  })
  
  console.log(fullName.value) // 输出：张三
  
  // 修改依赖项会触发计算属性重新计算
  firstName.value = '李'
  console.log(fullName.value) // 输出：李三
  ```

  在模板中使用：  
  ```vue
  <template>
    <div>
      <p>全名：{{ fullName }}</p>
      <input v-model="firstName" placeholder="姓" />
      <input v-model="lastName" placeholder="名" />
    </div>
  </template>
  ```

### 可写的计算属性
  默认情况下，计算‌‌‌属性是只读的。但有时我们可能需⁢⁢⁢要同时设置计算属性的值，这时可‏‏‏以使用可写的计算属性：    ‍‍‍   
  ```javascript
  import { ref, computed } from 'vue'
  
  const firstName = ref('张')
  const lastName = ref('三')
  
  const fullName = computed({
    // getter
    get() {
      return firstName.value + lastName.value
    },
    // setter
    set(newValue) {
      // 假设 newValue 的格式为 '姓名'
      const parts = newValue.split('')
      firstName.value = parts[0] || ''
      lastName.value = parts.slice(1).join('') || ''
    }
  })
  
  // 使用 setter
  fullName.value = '李四' // 会调用 setter
  
  console.log(firstName.value) // 输出：李
  console.log(lastName.value) // 输出：四
  ```

# 侦听器 (watch, watchEffect)
  侦听器用于在数据变化时执行副作用操作，如异步请求、DOM 操作等。Vue3 提供了两种侦听器 API：watch 和 watchEffect。  

## watch
  watch 用于侦听一个或多个响应式数据源，并在数据源变化时调用回调函数：  
  ```javascript
  import { ref, watch } from 'vue'
  
  const question = ref('')
  const answer = ref('等待提问...')
  
  // 侦听单个数据源
  watch(question, async (newQuestion, oldQuestion) => {
    if (newQuestion.trim() === '') {
      answer.value = '等待提问...'
    } else {
      answer.value = '思考中...'
      try {
        // 模拟 API 请求
        const response = await fetch('https://api.example.com/answer', {
          method: 'POST',
          body: JSON.stringify({ question: newQuestion })
        })
        const data = await response.json()
        answer.value = data.answer
      } catch (error) {
        answer.value = '抱歉，出错了'
      }
    }
  })
  ```

  侦听多个数据源：  
  ```javascript
  const firstName = ref('张')
  const lastName = ref('三')
  
  // 侦听多个数据源，使用数组
  watch([firstName, lastName], ([newFirst, newLast], [oldFirst, oldLast]) => {
    console.log('姓名变化：', oldFirst + oldLast, '->', newFirst + newLast)
  })
  ```

  侦听响应式‌对象‌的属性： 
  ```javascript
  import { reactive, watch } from 'vue'
  
  const user = reactive({
    name: '张三',
    age: 30
  })
  
  // 使用 getter 函数侦听对象的属性
  watch(
    () => user.name,
    (newName, oldName) => {
      console.log('用户名变化：', oldName, '->', newName)
    }
  )
  
  // 侦听整个对象
  watch(user, (newUser, oldUser) => {
    // 注意：newUser 和 oldUser 是同一个对象引用！
    console.log('用户对象变化')
  }, { deep: true }) // 使用 deep 选项进行深度侦听
  ```

  watch 选项：  
  ```javascript
  watch(source, callback, {
    immediate: true, // 立即执行一次回调
    deep: true,      // 深度侦听对象内部值的变化
    flush: 'post',   // 指定回调的刷新时机（'pre'、'post' 或 'sync'）
    onTrack(e) {     // 调试钩子，当依赖被追踪时调用
      console.log('依赖被追踪', e)
    },
    onTrigger(e) {   // 调试钩子，当依赖触发回调时调用
      console.log('依赖触发回调', e)
    }
  })
  ```

## watchEffect
  watchEffect 会自动追踪其依赖，并在依赖变化时重新执行：  
  ```javascript
  import { ref, watchEffect } from 'vue'
  
  const count = ref(0)
  const message = ref('Hello')
  
  watchEffect(() => {
    console.log(`Count: ${count.value}, Message: ${message.value}`)
  })
  // 立即输出：Count: 0, Message: Hello
  
  // 修改依赖会触发 watchEffect 回调
  count.value++
  // 输出：Count: 1, Message: Hello
  
  message.value = 'World'
  // 输出：Count: 1, Message: World
  ```

  watchEffect 与 watch 的区别：  
  1. 依赖追踪：
     - watchEffect 自动追踪回调中使用的所有响应式依赖
     - watch 需要明确指定要侦听的数据源
  2. 回调参数：
     - watchEffect 的回调没有参数，无法直接获取旧值
     - watch 的回调接收新值和旧值作为参数
  3. 执行时机：
     - watchEffect 默认会立即执行一次
     - watch 默认只在数据源变化时执行，除非设置 immediate: true

## 停止侦听器
  watch 和 watchEffect 都会返回一个停止侦听的函数：  
  ```javascript
  const stopWatch = watch(source, callback)
  const stopEffect = watchEffect(() => {})
  
  // 停止侦听
  stopWatch()
  stopEffect()
  ```

  在组件中，‌‌‌侦听器会在组件卸⁢⁢载⁢时自动停止。但‏‏如果‏在组件外部创‍‍建侦听‍器，需要手⁡⁡动停止以防止⁡内存泄漏。  

# 组合式函数（Composables）
  组合式函数是一种‌‌‌利用 Vue 的组合式 API⁢⁢⁢ 来封装和复用有状态逻辑的‏‏‏函数。它们是 Vue3 中实现逻辑‍‍‍复用的主要方式，替代了 Vue⁡⁡⁡2 中的 mixins。  

## 什么是组合式函数
  组合式函数是一个利用 Vue 组合式 API 来封装逻辑的函数，通常以 use 开头（例如 useCounter、useFetch）。它可以：  
  - 封装响应式状态
  - 封装方法和计算属性
  - 封装生命周期钩子
  - 返回需要暴露的状态和方法

## 创建组合式函数
  下面是一个‌‌‌简‌‌单的‌‌‌‌‌计数器组合⁢⁢式⁢函数：  
  ```javascript
  // useCounter.js
  import { ref } from 'vue'
  
  export function useCounter(initialValue = 0) {
    const count = ref(initialValue)
    
    function increment() {
      count.value++
    }
    
    function decrement() {
      count.value--
    }
    
    function reset() {
      count.value = initialValue
    }
    
    return {
      count,
      increment,
      decrement,
      reset
    }
  }
  ```

  在组件中使‌‌用：  
  ```vue
  <script setup>
  import { useCounter } from './composables/useCounter'
  
  const { count, increment, decrement, reset } = useCounter(10)
  </script>
  <template>
    <div>
      <p>Count: {{ count }}</p>
      <button @click="decrement">-</button>
      <button @click="reset">Reset</button>
      <button @click="increment">+</button>
    </div>
  </template>
  ```

## 组合式函数示例
### 封装异步数据获取逻辑
  ```javascript
    // useFetch.js
  import { ref, isRef, unref, watchEffect } from 'vue'
  
  export function useFetch(url) {
    const data = ref(null)
    const error = ref(null)
    const loading = ref(false)
    
    async function fetchData() {
      loading.value = true
      error.value = null
      
      try {
        // 如果 url 是 ref，则获取其值
        const response = await fetch(unref(url))
        if (!response.ok) {
          throw new Error(`请求失败: ${response.status}`)
        }
        data.value = await response.json()
      } catch (e) {
        error.value = e.message || '请求出错'
      } finally {
        loading.value = false
      }
    }
    
    // 如果 url 是 ref，当它变化时重新获取数据
    if (isRef(url)) {
      watchEffect(fetchData)
    } else {
      fetchData()
    }
    
    return { data, error, loading, refetch: fetchData }
  }
  ```

  在组件中使‌‌用：      
  ```vue
  <script setup>
  import { ref } from 'vue'
  import { useFetch } from './composables/useFetch'
  
  const userId = ref(1)
  const url = computed(() => `https://jsonplaceholder.typicode.com/users/${userId.value}`)
  
  const { data: user, error, loading, refetch } = useFetch(url)
  
  function nextUser() {
    userId.value++
  }
  </script>
  <template>
    <div>
      <p v-if="loading">加载中...</p>
      <p v-else-if="error">错误: {{ error }}</p>
      <div v-else-if="user">
        <h3>{{ user.name }}</h3>
        <p>Email: {{ user.email }}</p>
        <p>Phone: {{ user.phone }}</p>
      </div>
      <button @click="nextUser">下一个用户</button>
      <button @click="refetch">重新获取</button>
    </div>
  </template>
  ```

### 封装本地存储逻辑
  ```javascript
  // useLocalStorage.js
  import { ref, watch } from 'vue'
  
  export function useLocalStorage(key, defaultValue) {
    // 尝试从本地存储获取值，如果不存在则使用默认值
    const storedValue = localStorage.getItem(key)
    const value = ref(storedValue ? JSON.parse(storedValue) : defaultValue)
    
    // 当值变化时更新本地存储
    watch(value, (newValue) => {
      localStorage.setItem(key, JSON.stringify(newValue))
    }, { deep: true })
    
    // 提供清除方法
    function removeItem() {
      localStorage.removeItem(key)
      value.value = defaultValue
    }
    
    return { value, removeItem }
  }
  ```

  在组件中使‌‌用：   
  ```vue
  <script setup>
  import { useLocalStorage } from './composables/useLocalStorage'
  
  const { value: theme, removeItem: resetTheme } = useLocalStorage('theme', 'light')
  const { value: user, removeItem: resetUser } = useLocalStorage('user', { name: '', email: '' })
  
  function toggleTheme() {
    theme.value = theme.value === 'light' ? 'dark' : 'light'
  }
  </script>

  <template>
    <div :class="theme">
      <h2>当前主题: {{ theme }}</h2>
      <button @click="toggleTheme">切换主题</button>
      <button @click="resetTheme">重置主题</button>
      
      <div class="user-form">
        <input v-model="user.name" placeholder="姓名" />
        <input v-model="user.email" placeholder="邮箱" />
        <button @click="resetUser">重置用户</button>
      </div>
    </div>
  </template>

  <style scoped>
  .light {
    background-color: #f8f9fa;
    color: #212529;
  }
  
  .dark {
    background-color: #212529;
    color: #f8f9fa;
  }
  
  .user-form {
    margin-top: 20px;
  }
  
  input {
    margin-right: 10px;
    padding: 5px;
  }
  
  button {
    margin-right: 10px;
    padding: 5px 10px;
  }
  </style>
  ```

## 组合式函数的最佳实践
  1. 命名约定：使用 use 前缀命名组合式函数，如 useCounter、useFetch。
  2. 返回值：返回一个包含响应式状态和方法的对象，使用解构赋值获取。
  3. 参数设计：
     - 接受配置参数，提供默认值
     - 支持响应式参数（ref、computed）
  4. 使用 unref 或 toValue（Vue 3.3+）处理可能是 ref 的参数。
  5. 副作用清理：在组合式函数中注册的副作用（如事件监听器、定时器）应该在适当的时候清理。
     ```javascript
      import { onUnmounted } from 'vue'
      
      export function useEventListener(target, event, callback) {
        // 如果 target 是 ref，获取其值
        const element = unref(target)
        
        element.addEventListener(event, callback)
        
        // 在组件卸载时移除事件监听器
        onUnmounted(() => {
          element.removeEventListener(event, callback)
        })
      }
     ```
  6. 组合多个组合式函数：组合式函数可以调用其他组合式函数，构建更复杂的逻辑。
     ```javascript
      export function useUserProfile(userId) {
        // 复用 useFetch 组合式函数
        const url = computed(() => `https://api.example.com/users/${userId.value}`)
        const { data, loading, error, refetch } = useFetch(url)
        
        // 复用 useLocalStorage 组合式函数
        const { value: savedPreferences } = useLocalStorage(`user-${userId.value}-prefs`, {})
        
        // 添加额外的逻辑
        const isAdmin = computed(() => data.value?.role === 'admin')
        
        return {
          user: data,
          loading,
          error,
          refetch,
          savedPreferences,
          isAdmin
        }
      }
     ```

# Provide/Inject 与依赖注入
  Provide/In‌‌‌ject 是 Vue 中实现跨层级组件⁢⁢⁢通信的机制，特别适合在深层嵌套的组件中‏‏‏共享数据，而不必通过每一层组件传递 pr‍‍‍ops。       

## 基本用法
  在父组件中提供数据：  
  ```javascript
  import { provide, ref } from 'vue'
  
  export default {
    setup() {
      const theme = ref('light')
      
      // 提供值给后代组件
      provide('theme', theme)
      
      function toggleTheme() {
        theme.value = theme.value === 'light' ? 'dark' : 'light'
      }
      
      return { toggleTheme }
    }
  }
  ```

  在后代组件中注入数据：  
  ```javascript
  import { inject } from 'vue'
  
  export default {
    setup() {
      // 注入祖先组件提供的值
      const theme = inject('theme')
      
      return { theme }
    }
  }
  ```

  使用 <script setup> 语法：  
  ```vue
  <!-- 父组件 -->
  <script setup>
  
  const theme = ref('light')
  provide('theme', theme)
  
  function toggleTheme() {
    theme.value = theme.value === 'light' ? 'dark' : 'light'
  }
  </script>

  <!-- 子组件 -->
  <script setup>
  import { inject } from 'vue'
  
  const theimport { provide, ref } from 'vue'
  me = inject('theme')
  </script>
  ```

## 提供与注入响应式数据
  当提供响应‌‌‌式数据时，注入的组件⁢⁢⁢也会创建对该数据的响应‏式‏‏引用。这意味着父‍‍‍组件中的更新会传递给⁡⁡⁡子组件，保持数据的同步。  
  ```vue
  <!-- 父组件 -->
  <script setup>
  import { provide, ref } from 'vue'
  
  const count = ref(0)
  provide('count', count) // 提供响应式引用
  
  function increment() {
    count.value++
  }
  </script>

  <template>
    <div>
      <p>父组件计数: {{ count }}</p>
      <button @click="increment">增加</button>
      <child-component />
    </div>
  </template>
  <!-- 子组件 -->
  <script setup>
  import { inject } from 'vue'
  
  const count = inject('count') // 获取响应式引用
  </script>

  <template>
    <div>
      <p>子组件计数: {{ count }}</p>
    </div>
  </template>
  ```

## 提供方法
  除了提供数‌‌‌据，还可以提供⁢方⁢法⁢，让子组件‏能够‏修改‏父组件‍中的状‍态：  
  ```vue
  <!-- 父组件 -->
  <script setup>
  import { provide, ref } from 'vue'
  
  const count = ref(0)
  
  function increment() {
    count.value++
  }
  
  function decrement() {
    count.value--
  }
  
  // 提供数据和方法
  provide('count', count)
  provide('increment', increment)
  provide('decrement', decrement)
  </script>
  <!-- 子组件 -->
  <script setup>
  import { inject } from 'vue'
  
  const count = inject('count')
  const increment = inject('increment')
  const decrement = inject('decrement')
  </script>

  <template>
    <div>
      <p>计数: {{ count }}</p>
      <button @click="decrement">-</button>
      <button @click="increment">+</button>
    </div>
  </template>
  ```

## 使用 Symbol 作为注入名
  为了避免命名冲突，可以使用 Symbol 作为注入名：  
  ```// keys.js
  export const themeKey = Symbol('theme')
  export const userKey = Symbol('user')
  javascript
  ```

  ```vue
  <!-- 父组件 -->
  <script setup>
  import { provide, ref } from 'vue'
  import { themeKey } from './keys'
  
  const theme = ref('light')
  provide(themeKey, theme)
  </script>

  <!-- 子组件 -->
  <script setup>
  import { inject } from 'vue'
  import { themeKey } from './keys'
  
  const theme = inject(themeKey)
  </script>
  ```

## 设置默认值
  注入值时可以‌设置‌默认值，当祖先组件⁢没有提供对应的值⁢时使用:  
  ```javascript
  // 基本默认值
  const theme = inject('theme', 'light')
  
  // 使用工厂函数创建默认值
  const theme = inject('theme', () => 'light')
  ```

## 使用 readonly 防止修改
  为了防止注入的组件直接修改提供的状态，可以使用 readonly 包装提供的值：  
  ```javascript
  import { provide, ref, readonly } from 'vue'
  
  const count = ref(0)
  
  // 提供只读版本
  provide('count', readonly(count))
  
  // 提供修改方法
  provide('increment', () => { count.value++ })
  ```

# 练习题
## 练习 1：创建一个使用计算属性和侦听器的购物车组件
  创建一个购‌‌‌物车组件，使用计⁢⁢算⁢属性计算总价和‏‏商品‏数量，使用侦‍‍听器在‍商品变化时⁡⁡保存到⁡本地存储。  

  答案：  
  ```vue
  <script setup>
  import { ref, computed, watch } from 'vue'
  
  // 商品列表
  const items = ref(JSON.parse(localStorage.getItem('cart-items') || '[]'))
  
  // 计算属性：总价
  const totalPrice = computed(() => {
    return items.value.reduce((total, item) => {
      return total + item.price * item.quantity
    }, 0).toFixed(2)
  })
  
  // 计算属性：商品总数
  const totalItems = computed(() => {
    return items.value.reduce((count, item) => {
      return count + item.quantity
    }, 0)
  })
  
  // 侦听器：保存到本地存储
  watch(items, (newItems) => {
    localStorage.setItem('cart-items', JSON.stringify(newItems))
  }, { deep: true })
  
  // 添加商品
  function addItem(item) {
    const existingItem = items.value.find(i => i.id === item.id)
    
    if (existingItem) {
      existingItem.quantity++
    } else {
      items.value.push({
        ...item,
        quantity: 1
      })
    }
  }
  
  // 减少商品数量
  function decreaseQuantity(itemId) {
    const index = items.value.findIndex(item => item.id === itemId)
    
    if (index !== -1) {
      if (items.value[index].quantity > 1) {
        items.value[index].quantity--
      } else {
        removeItem(itemId)
      }
    }
  }
  
  // 移除商品
  function removeItem(itemId) {
    items.value = items.value.filter(item => item.id !== itemId)
  }
  
  // 清空购物车
  function clearCart() {
    items.value = []
  }
  
  // 示例商品
  const products = [
    { id: 1, name: '手机', price: 3999 },
    { id: 2, name: '笔记本电脑', price: 5999 },
    { id: 3, name: '耳机', price: 299 },
    { id: 4, name: '平板电脑', price: 2999 }
  ]
  </script>

  <template>
    <div class="shopping-cart">
      <h2>购物车</h2>
      
      <div class="products">
        <h3>商品列表</h3>
        <div class="product-list">
          <div v-for="product in products" :key="product.id" class="product-item">
            <span>{{ product.name }} - ¥{{ product.price }}</span>
            <button @click="addItem(product)">添加到购物车</button>
          </div>
        </div>
      </div>
      
      <div class="cart">
        <h3>购物车内容 ({{ totalItems }} 件商品)</h3>
        
        <div v-if="items.length === 0" class="empty-cart">
          购物车为空
        </div>
        
        <div v-else>
          <div v-for="item in items" :key="item.id" class="cart-item">
            <div class="item-info">
              <span>{{ item.name }}</span>
              <span>¥{{ item.price }} × {{ item.quantity }}</span>
            </div>
            <div class="item-actions">
              <button @click="decreaseQuantity(item.id)">-</button>
              <span>{{ item.quantity }}</span>
              <button @click="addItem(item)">+</button>
              <button @click="removeItem(item.id)" class="remove-btn">删除</button>
            </div>
  ```

## 练习 2：创建一个主题切换的组合式函数
  创建一个 useTheme 组合式函数，实现主题切换功能，并在多个组件中复用。  

  答案：  
  ```javascript
  // composables/useTheme.js
  import { ref, watch, onMounted, provide, inject } from 'vue'
  
  // Symbol 用于提供/注入的键
  export const themeKey = Symbol('theme')
  
  export function useTheme() {
    // 尝试注入现有主题，如果不存在则创建新的
    const theme = inject(themeKey, null) || ref(localStorage.getItem('theme') || 'light')
    
    // 如果是新创建的主题，则提供给后代组件
    if (!inject(themeKey, null)) {
      provide(themeKey, theme)
    }
    
    // 切换主题
    function toggleTheme() {
      theme.value = theme.value === 'light' ? 'dark' : 'light'
    }
    
    // 设置特定主题
    function setTheme(newTheme) {
      if (newTheme === 'light' || newTheme === 'dark') {
        theme.value = newTheme
      } else {
        console.warn('主题只能设置为 "light" 或 "dark"')
      }
    }
    
    // 监听主题变化，更新 localStorage 和 document 的 class
    watch(theme, (newTheme) => {
      // 保存到 localStorage
      localStorage.setItem('theme', newTheme)
      
      // 更新 document 的 class
      if (newTheme === 'dark') {
        document.documentElement.classList.add('dark-theme')
        document.documentElement.classList.remove('light-theme')
      } else {
        document.documentElement.classList.add('light-theme')
        document.documentElement.classList.remove('dark-theme')
      }
    }, { immediate: true })
    
    // 组件挂载时应用主题
    onMounted(() => {
      // 初始应用主题
      if (theme.value === 'dark') {
        document.documentElement.classList.add('dark-theme')
        document.documentElement.classList.remove('light-theme')
      } else {
        document.documentElement.classList.add('light-theme')
        document.documentElement.classList.remove('dark-theme')
      }
    })
    
    return {
      theme,
      toggleTheme,
      setTheme,
      isDark: computed(() => theme.value === 'dark')
    }
  }
  ```

  在 App 组件中使用：  

## 练习 3：创建一个使用 Provide/Inject 的多级菜单组件
  创建一个多级菜‌‌‌单组件，使用 Provid⁢⁢⁢e/Inject 在各级菜‏‏‏单之间共享状态。     ‍‍‍ 

  答案：  
  ```vue

  import Menu from './components/Menu.vue'

        {
          id: 23,
          label: '配件',
          link: '/products/accessories',
          children: [
            { id: 231, label: '耳机', link: '/products/accessories/headphones' },
            { id: 232, label: '充电器', link: '/products/accessories/chargers' }

        }

    {
      id: 3,
      label: '关于我们',

    },
    {

      label: '联系我们',
      link: '/contact'
    }

  // 当前激活的菜单项
  const activeItemId = ref(1)
  
  // 提供菜单状态给所有子组件
  provide('activeItemId', activeItemId)
  
  // 设置激活项的方法
  function setActiveItem(id) {
    activeItemId.value = id
  }
  provide('setActiveItem', setActiveItem)
  
  // 展开的子菜单 ID 集合
  const expandedMenus = ref(new Set())
  
  // 切换子菜单展开状态
  function toggleSubmenu(id) {
    if (expandedMenus.value.has(id)) {
      expandedMenus.value.delete(id)
    } else {
      expandedMenus.value.add(id)
    }
  }
  provide('expandedMenus', expandedMenus)
  provide('toggleSubmenu', toggleSubmenu)
  </script>
  <template>
  ```

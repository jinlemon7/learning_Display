在本章中，你将学习‌ V‌‌‌‌‌ue3 中各种组件通信的方式，⁢掌握如何在不同层⁢⁢⁢⁢⁢级的组件之间传递数据和‏事件。通过这些知识，你将‏‏‏‏‏能够构‍建出组件之间协调工作的复杂应用，使⁡组件‍‍‍‍‍能够高效地共享和交换信息       

# 父子组件通信
  父子组件是‌‌‌‌‌‌最常见的组件⁢⁢⁢关⁢系⁢，⁢Vue‏‏‏3 ‏提供‏了几‏‍‍‍种方式‍来实现‍它⁡⁡⁡们之间⁡‍的通信  

## Props 向下传递数据
  父组件通过 ‌‌pr‌‌‌‌ops 向子组件传⁢⁢递数据是最基本的⁢⁢⁢⁢通信方‏‏式：       
  ```vue
  <!-- 父组件 -->
  <script setup>
  import { ref } from 'vue'
  import ChildComponent from './ChildComponent.vue'
  
  const message = ref('来自父组件的消息')
  const userInfo = ref({
    name: '张三',
    age: 30
  })
  </script>
  
  <template>
    <div>
      <h2>父组件</h2>
      <ChildComponent 
        :message="message" 
        :user-info="userInfo" 
      />
    </div>
  </template>
  ```

  ```vue
  <!-- 子组件 ChildComponent.vue -->
  <script setup>
  const props = defineProps({
    message: String,
    userInfo: Object
  })
  </script>

  <template>
    <div>
      <h3>子组件</h3>
      <p>接收到的消息: {{ message }}</p>
      <p>用户名: {{ userInfo.name }}</p>
      <p>年龄: {{ userInfo.age }}</p>
    </div>
  </template>
  ```

## 事件向上传递数据
  子组件可以通过触发事件向父组件传递数据：  
  ```vue
  <!-- 子组件 ChildComponent.vue -->
  <script setup>
  import { ref } from 'vue'
  
  const inputValue = ref('')
  const emit = defineEmits(['update-data', 'submit'])
  
  function sendDataToParent() {
    emit('update-data', inputValue.value)
  }
  
  function submitForm() {
    emit('submit', {
      value: inputValue.value,
      timestamp: Date.now()
    })
  }
  </script>

  <template>
    <div>
      <input v-model="inputValue" placeholder="输入内容" />
      <button @click="sendDataToParent">发送数据到父组件</button>
      <button @click="submitForm">提交表单</button>
    </div>
  </template>
  ```

  ``` vue
  <!-- 父组件 -->
  <script setup>
  import { ref } from 'vue'
  import ChildComponent from './ChildComponent.vue'
  
  const receivedData = ref('')
  
  function handleUpdateData(data) {
    receivedData.value = data
    console.log('从子组件接收到数据:', data)
  }
  
  function handleSubmit(formData) {
    console.log('表单提交:', formData)
    // 处理表单提交逻辑
  }
  </script>
  <template>
    <div>
      <h2>父组件</h2>
      <p v-if="receivedData">接收到的数据: {{ receivedData }}</p>
      
      <ChildComponent 
        @update-data="handleUpdateData"
        @submit="handleSubmit"
      />
    </div>
  </template>
  ```

## v-model 实现双向绑定
  Vue3 中的 v-model 是 props 和事件的语法糖，可以实现父子组件之间的双向数据绑定：  
  ```vue
  <!-- 父组件 -->
  <script setup>
  import { ref } from 'vue'
  import CustomInput from './CustomInput.vue'
  
  const username = ref('')
  </script>
  
  <template>
    <div>
      <h2>父组件</h2>
      <p>用户名: {{ username }}</p>
      
      <CustomInput v-model="username" />
    </div>
  </template>
  ```

  ```vue
  <!-- CustomInput.vue -->
  <script setup>
  defineProps({
    modelValue: String
  })
  
  const emit = defineEmits(['update:modelValue'])
  
  function updateValue(event) {
    emit('update:modelValue', event.target.value)
  }
  </script>
  <template>
    <input 
      :value="modelValue"
      @input="updateValue"
      class="custom-input"
    />
  </template>
  <style scoped>
  .custom-input {
    padding: 8px;
    border: 1px solid #ddd;
    border-radius: 4px;
  }
  </style>
  ```

  在 Vue3 中，你还可以使用多个 v-model 绑定：  
  ```vue
  <!-- 父组件 -->
  <template>
    <UserForm
      v-model:first-name="firstName"
      v-model:last-name="lastName"
    />
  </template>
  ```

  ```vue
  <!-- UserForm.vue -->
  <script setup>
  defineProps({
    firstName: String,
    lastName: String
  })
  
  const emit = defineEmits(['update:firstName', 'update:lastName'])
  </script>
  <template>
    <input 
      :value="firstName"
      @input="emit('update:firstName', $event.target.value)"
    />
    <input 
      :value="lastName"
      @input="emit('update:lastName', $event.target.value)"
    />
  </template>
  ```

# 兄弟组件通信
  兄弟组件之‌‌‌‌‌‌间没有直接的通⁢信⁢方⁢式，但⁢⁢⁢可以‏通过‏共同‏的父组‍件或‏‏‏其‍他方式来实‍⁡现通信⁡。  

## 通过父组件中转
  最简单的方式是通过共同的父组件作为中间人：  
  ```vue
  <!-- 父组件 -->
  <script setup>
  import { ref } from 'vue'
  import SiblingA from './SiblingA.vue'
  import SiblingB from './SiblingB.vue'
  
  const sharedData = ref('初始数据')
  
  function updateData(newData) {
    sharedData.value = newData
  }
  </script>
  <template>
    <div>
      <h2>父组件</h2>
      <SiblingA 
        :shared-data="sharedData" 
        @update-data="updateData"
      />
      <SiblingB :shared-data="sharedData" />
    </div>
  </template>
  ```

  ```vue
  <!-- SiblingA.vue -->
  <script setup>
  import { ref } from 'vue'
  
  defineProps({
    sharedData: String
  })
  
  const emit = defineEmits(['update-data'])
  const inputData = ref('')
  
  function sendData() {
    emit('update-data', inputData.value)
    inputData.value = ''
  }
  </script>
  <template>
    <div class="sibling">
      <h3>组件 A</h3>
      <p>共享数据: {{ sharedData }}</p>
      <input v-model="inputData" placeholder="输入新数据" />
      <button @click="sendData">发送数据</button>
    </div>
  </template>
  ```

  ```vue
  <!-- SiblingB.vue -->
  <script setup>
  defineProps({
    sharedData: String
  })
  </script>
  <template>
    <div class="sibling">
      <h3>组件 B</h3>
      <p>接收到的共享数据: {{ sharedData }}</p>
    </div>
  </template>
  ```

## 使用事件总线（mitt）
  在 Vue3 中，官方移除了内置的事件总线，但我们可以使用第三方库如 mitt 来实现：  

  首先安装 ‌‌mitt：     
  ```bash
  npm install mitt
  ```

  创建一个事‌‌件‌‌总线实例：   
  ```javascript
  // eventBus.js
  import mitt from 'mitt'
  const emitter = mitt()
  export default emitter
  ```

  在组件中使用：  
  ```vue
  <!-- SiblingA.vue -->
  <script setup>
  import { ref } from 'vue'
  import emitter from '../eventBus'
  
  const message = ref('')
  
  function sendMessage() {
    if (message.value) {
      emitter.emit('message-from-a', message.value)
      message.value = ''
    }
  }
  </script>
  <template>
    <div>
      <h3>组件 A</h3>
      <input v-model="message" placeholder="发送消息" />
      <button @click="sendMessage">发送到组件 B</button>
    </div>
  </template>
  ```

  ```vue
  <!-- SiblingB.vue -->
  <script setup>
  import { ref, onMounted, onUnmounted } from 'vue'
  import emitter from '../eventBus'
  
  const receivedMessages = ref([])
  
  // 监听事件
  function onMessageFromA(message) {
    receivedMessages.value.push(message)
  }
  
  onMounted(() => {
    emitter.on('message-from-a', onMessageFromA)
  })
  
  onUnmounted(() => {
    emitter.off('message-from-a', onMessageFromA)
  })
  </script>
  <template>
    <div>
      <h3>组件 B</h3>
      <p v-if="receivedMessages.length === 0">暂无消息</p>
      <ul v-else>
        <li v-for="(msg, index) in receivedMessages" :key="index">
          {{ msg }}
        </li>
      </ul>
    </div>
  </template>
  ```

# 跨层级组件通信
  当组件层级‌‌‌‌较深时，通过 pr⁢⁢⁢⁢ops 逐层传递数‏‏‏‏据会变得繁琐，Vu‍‍‍‍e3 提供了几种方⁡⁡⁡⁡式来解决这个问题。  

## 使用 Provide 和 Inject
  provide 和 inject 是 Vue 提供的依赖注入机制，可以让深层嵌套的组件访问上层组件提供的数据，而不必通过 props 层层传递：  

  ```vue
  <!-- 祖先组件 -->
  <script setup>
  import { ref, provide } from 'vue'
  
  const theme = ref('light')
  const user = ref({
    name: '张三',
    role: 'admin'
  })
  
  // 提供响应式数据
  provide('theme', theme)
  provide('user', user)
  
  function toggleTheme() {
    theme.value = theme.value === 'light' ? 'dark' : 'light'
  }
  </script>
  <template>
    <div :class="theme">
      <h1>应用</h1>
      <button @click="toggleTheme">切换主题</button>
      <slot></slot>
    </div>
  </template>
  ```

  ```vue
  <!-- 深层嵌套的后代组件 -->
  <script setup>
  import { inject } from 'vue'
  
  // 注入数据，可以提供默认值
  const theme = inject('theme', 'light')
  const user = inject('user', { name: '游客', role: 'guest' })
  </script>
  <template>
    <div class="component">
      <h3>深层嵌套组件</h3>
      <p>当前主题: {{ theme }}</p>
      <p>当前用户: {{ user.name }} ({{ user.role }})</p>
    </div>
  </template>
  ```

## 使用 Vuex 或 Pinia 进行状态管理
  对于更复杂‌‌‌‌的应用，可以使⁢⁢用⁢状⁢态管理库如‏‏ P‏in‏ia（‍‍Vue‍3 推‍荐⁡⁡）来管理⁡共享状态⁡：

  ```javascript
      name: 'Eduardo'

      doubleCount: (state) => state.count * 2

    actions: {
      increment() {
        this.count++
      },
      async fetchData() {
        // 异步操作
        const data = await api.getData()
        this.count = data.count
      }
    }
  })
  ```

  在任何组件中使用：  

# Provide 和 Inject
  我们已经在跨层级通信中简单介绍了 provide 和 inject，这里我们来深入了解它们的更多用法。  

## 提供和注入响应式数据
  当提供响应‌‌‌‌式数据时，建议⁢⁢直⁢接⁢提供 re‏‏f ‏或 ‏rea‍‍cti‍ve ‍对⁡⁡象，而不⁡是它们的⁡值：  
  ```vue
  <script setup>
  import { ref, provide, readonly } from 'vue'
  
  const count = ref(0)
  
  // 提供响应式数据
  provide('count', count) // 后代组件可以修改这个值
  
  // 如果不希望后代组件修改这个值，可以提供只读版本
  provide('readonlyCount', readonly(count))
  
  function increment() {
    count.value++
  }
  </script>
  ```

## 提供和注入方法
  除了数据，‌‌‌‌你还可以提供方法给⁢⁢⁢⁢后代组件使用  
  ```vue
  <script setup>
  import { provide } from 'vue'
  
  function doSomething(value) {
    console.log('做一些事情:', value)
    // 处理逻辑...
  }
  
  provide('doSomething', doSomething)
  </script>
  ```

  ```vue
  <script setup>
  import { inject } from 'vue'
  
  const doSomething = inject('doSomething')
  
  function handleClick() {
    if (doSomething) {
      doSomething('来自后代组件的数据')
    }
  }
  </script>
  <template>
    <button @click="handleClick">调用祖先组件的方法</button>
  </template>
  ```

## 使用 Symbol 作为 key
  为了避免命‌‌‌‌名冲突，可以使用⁢⁢⁢ ⁢Symbol ‏‏‏作为‏ provi‍‍‍de/‍injec⁡⁡⁡t 的 ⁡key：  
  ```javascript
  // keys.js
  export const themeSymbol = Symbol('theme')
  export const userSymbol = Symbol('user')
  ```

  ```vue
  <script setup>
  import { provide, ref } from 'vue'
  import { themeSymbol } from './keys'
  
  const theme = ref('dark')
  provide(themeSymbol, theme)
  </script>
  ```

  ```vue
  <script setup>
  import { inject } from 'vue'
  import { themeSymbol } from './keys'
  
  const theme = inject(themeSymbol)
  </script>
  ```

# 事件总线（mitt）
  我们在兄弟‌‌‌‌组件通信中已经⁢⁢简⁢单⁢介绍了 m‏‏it‏t ‏事件总‍‍线，这‍里我们‍来⁡⁡看一些更高级⁡的用⁡法。  

## 全局事件监听和移除
  ```javascript
  import emitter from './eventBus'
  
  // 监听所有事件
  emitter.on('*', (type, payload) => {
    console.log('事件类型:', type)
    console.log('事件数据:', payload)
  })
  
  // 移除特定事件的所有监听器
  emitter.off('user-login')
  
  // 移除所有事件的所有监听器
  emitter.all.clear()
  ```

## 在组合式函数中使用事件总线
  可以创建一个组合式函数来封装事件总线的使用：  
  ```javascript
  // useEventBus.js
  import { onUnmounted } from 'vue'
  import emitter from './eventBus'
  
  export function useEventBus() {
    const on = (event, callback) => {
      emitter.on(event, callback)
    }
    
    const emit = (event, payload) => {
      emitter.emit(event, payload)
    }
    
    const off = (event, callback) => {
      emitter.off(event, callback)
    }
    
    // 组件卸载时自动清理监听器
    onUnmounted(() => {
      // 如果需要，可以在这里移除特定的监听器
    })
    
    return { on, emit, off }
  }
  ```

  在组件中使用：  
  ```vue
  <script setup>
  import { ref } from 'vue'
  import { useEventBus } from '../composables/useEventBus'
  
  const { on, emit } = useEventBus()
  const messages = ref([])
  
  on('new-message', (message) => {
    messages.value.push(message)
  })
  
  function sendMessage() {
    emit('new-message', {
      text: '你好，世界！',
      timestamp: new Date().toLocaleTimeString()
    })
  }
  </script>
  ```

## 练习 1：实现父子组件双向数据绑定
  创建一个自‌‌‌‌定义的表单组⁢件⁢，⁢支⁢持双向‏数据‏绑定‏，包‏‍含输入‍框和验证⁡‍功能‍。  

  答案：  
  ```vue
  <!-- FormInput.vue -->
  <script setup>
  import { computed } from 'vue'
  
  const props = defineProps({
    modelValue: String,
    label: String,
    required: {
      type: Boolean,
      default: false
    }
  })
  
  const emit = defineEmits(['update:modelValue'])
  
  const isValid = computed(() => {
    if (props.required) {
      return !!props.modelValue && props.modelValue.trim() !== ''
    }
    return true
  })
  
  function updateValue(event) {
    emit('update:modelValue', event.target.value)
  }
  </script>
  <template>
    <div class="form-group">
      <label v-if="label">{{ label }}{{ required ? ' *' : '' }}</label>
      <input 
        :value="modelValue"
        @input="updateValue"
        :class="{ 'invalid': !isValid }"
      />
      <span v-if="!isValid" class="error-message">
        此字段为必填项
      </span>
    </div>
  </template>
  <style scoped>
  .form-group {
    margin-bottom: 15px;
  }
  
  label {
    display: block;
    margin-bottom: 5px;
    font-weight: bold;
  }
  
  input {
    width: 100%;
    padding: 8px;
    border: 1px solid #ddd;
    border-radius: 4px;
  }
  
  input.invalid {
    border-color: #e74c3c;
  }
  
  .error-message {
    color: #e74c3c;
    font-size: 12px;
    margin-top: 5px;
    display: block;
  }
  </style>
  ```

  使用示例：‌‌   

  通过本章的学习，你已经掌握了‌‌‌ V‌ue3 中各种组件通信的方式，包括父子组件通信、兄弟⁢⁢⁢组件通信和跨层级⁢组件通信。  
  这些知识将帮助你构建出组件之间‏‏‏协调工作的复杂应用，使组件能‏够高效地共享和交换信息。在下‍‍‍一章中，我们将深入学习 Vue3 的组合‍式 API 基础⁡⁡⁡，这是 Vue3 最重要的新特性之一

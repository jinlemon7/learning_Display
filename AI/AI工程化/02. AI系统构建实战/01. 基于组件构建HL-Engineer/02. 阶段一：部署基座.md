 # 环境准备 (WSL2 与基础工具)
  首先，需要在Windows 11上安装WSL2，因为核心组件Hermes Agent需要在Linux环境下运行。  

  ```shell
  # 在 Windows PowerShell（管理员）中执行
  wsl --install -d Ubuntu-24.04
  ```

  重启电脑后，Ubuntu将自动启动。创建用户名和密码后，在WSL终端中更新系统并安装Git和GitHub CLI：  
  ```shell
  sudo apt update && sudo apt upgrade -y
  sudo apt install -y git gh
  ```

# 安装核心组件

## 安装 OpenClaw (指挥官)
  使用 openclaw 这个npm包来安装OpenClaw的命令行工具和运行时。  
  ```bash
  # 使用 nvm 安装 Node.js LTS 版本
  curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
  source ~/.bashrc
  nvm install --lts
  
  # 验证安装
  node --version
  npm --version
  ```

  ```bash
  # 全局安装 OpenClaw（如果该包不存在，可尝试安装 openclaw-cli）
  npm install -g openclaw
  ```

  ```bash
  # 运行配置向导，完成LLM设置和基础配置
  openclaw onboard
  ```

## 安装 Hermes Agent (实干家)
  使用官方提供的一键安装脚本来部署Hermes Agent，它会自行处理Python等复杂依赖。  
  ```bash
  curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
  ```

  ```bash
  # 通过交互式向导，完成初始配置
  hermes setup
  ```

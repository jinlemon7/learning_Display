# 忙输入模式设定
  详见  [队列 vs 中断 vs 转向（忙输入模式）](https://github.com/jinlemon7/learning_Display/blob/main/AI/AI工程化/02.%20AI组件/02.%20Hermes/05.%20即时通讯平台/01.%20消息网关.md#队列-vs-中断-vs-转向忙输入模式)  

  ```bash
  display:
    busy_input_mode: steer   # or queue, or interrupt (default)
    busy_ack_enabled: false   # set to false to suppress the ⚡/⏳/⏩ chat reply entirely
  ```

# 显示设置

由 Nous Research 开发的自我改进型 AI agent。它是唯一一款内置学习循环的agent——它从经验中创造技能，在使用过程中不断改进，自我引导以巩固知识，并在不同会话中构建不断加深的关于你的模型。  


# Hermes Agent 是什么？
  它不是依附于集成开发环境 (IDE) 的代码助手，也不是围绕单一 API 封装的聊天机器人。它是一个自主agent ，运行时间越长，功能越强大。它可以部署在任何你指定的环境中——无论是 5 美元的虚拟专用服务器 (VPS)、GPU 集群，还是闲置时几乎不产生任何成本的无服务器基础设施（例如 Daytona、Modal）。即使它运行在你从未通过 SSH 连接过的云虚拟机上，你也可以通过 Telegram 与它互动。它并不依赖于你的笔记本电脑。  

# 主要特点
  一个闭环学习系统 ——由智能体维护的记忆（附带定期提醒）、自主技能创建、使用过程中的技能自我改进、FTS5 跨会话回忆（结合 LLM 摘要）以及 Honcho 辩证用户建模  

  可在任何设备上运行，不仅限于笔记本电脑 ——支持 6 种终端后端：本地、Docker、SSH、Daytona、Singularity 和 Modal。Daytona 和 Modal 提供无服务器持久化功能——环境在空闲时会进入休眠状态，几乎不消耗任何资源。  

  随时随地，尽在掌握 ——CLI、Telegram、Discord、Slack、WhatsApp、Signal、Matrix、Mattermost、电子邮件、短信、钉钉、飞书、微商、BlueBubbles、Home Assistant、Microsoft Teams——通过一个网关即可访问 15 个以上的平台。  

  由模型训练专家构建 ——由 Nous Research 实验室创建，该实验室也是 Hermes、Nomos 和 Psyche 的开发者。可与 Nous Portal 、 OpenRouter 、OpenAI 或任何端点配合使用。  

  定时自动化 — 内置 cron 功能，可部署到任何平台  

  委托和并行化 — 为并行工作流生成隔离的子agent。通过 execute_code 进行程序化工具调用，可将多步骤管道合并为单个推理调用。  

  开放标准技能 ——与 agentskills.io 兼容。技能可移植、可共享，并通过技能中心由社区贡献。  

  完全网页控制 ——搜索、提取、浏览、视觉识别、图像生成、文本转语音  

  MCP 支持 — 连接到任何 MCP 服务器以扩展工具功能  

  研究就绪 ——支持批量处理、轨迹导出和使用 Atropos 进行强化学习训练。由 Nous Research 构建——该实验室正是 Hermes、Nomos 和 Psyche 模型背后的研发者。  

# 对于 LLM 和编码 agent
  机器可读的文档入口点：  
  - /llms.txt — 包含所有文档页面及其简短描述的精选索引。文件大小约为 17 KB，可安全加载到 LLM 上下文中。
  - /llms-full.txt — 所有文档页面合并成一个 Markdown 文件，以便一次性导入。约 1.8 MB。

  这两个文件分别解析为 /docs/llms.txt 和 /docs/llms-full.txt 。每次部署时都会重新生成。

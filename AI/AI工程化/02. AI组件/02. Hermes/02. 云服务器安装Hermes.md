# 运行一键安装脚本
  ```bash
  curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
  ```

# 安装完成后，重新加载配置终端：
  ```bash  
  source ~/.bashrc
  ```

# 检查是否安装完成
  ```bash
  hermes --version
  ```

# 环境健康检查并修复
  ```bash
  hermes doctor
  ```

# 配置LLM
  ```bash
  hermes model

  - 选择模型提供商: DeepSeek
  - 输入API key
  - 输入Base URL：https://api.deepseek.com/anthropic
  - 选择Default model：deepseek-v4-flash (via DeepSeek)
  ```

# 第一次对话
  ```bash
  hermes chat -q '帮我分析一下当前的开发环境——装了什么开发语言，什么包管理器，磁盘还剩多少'
  ```

  - hermes: 启动对话终端
  - hermes chat：启动对话终端
  - hermes chat -q：单次任务，直接返回。实现任务分发，多agent协作。
  - hermes chat -m：指定模型
  - hermes gateway：消息网关

# 连接消息平台机器人
  获得消息平台机器人的APP ID 和 APP Secret  
  ```bash
  hermes gateway setup
  ```

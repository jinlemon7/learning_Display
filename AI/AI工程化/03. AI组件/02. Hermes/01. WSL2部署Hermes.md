# 安装
  ```bash
  curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
  ```

# 运行配置向导
  交互式向导会引导你完成 provider、模型、工具等核心配置：  
  ```bash
  hermes setup
  ```
  - 配置模型key时界面不会显示

# 验证安装
  ```bash
  hermes version
  ```
  - 显示版本号即安装成功。如果提示 command not found，执行 source ~/.bashrc（或 ~/.zshrc）刷新 shell。

# 运行诊断
  首次使用前建议运行诊断，确认所有依赖正常：  
  ```bash
  hermes doctor
  ```

# 第一次对话
  直接运行 hermes 进入交互式 CLI：  
  ```bash
  hermes

  You: 你好，Hermes
  Hermes: 你好！我是 Hermes，你的 AI 助手。有什么我能帮你的吗？
  ```

# 连接消息平台机器人
  配置 消息平台机器人  
  ```bash
  hermes gateway setup
  ```
  - 连接QQ bot时消息连接方式使用pair配对

# 启动 Gateway
  前台运行（推荐 WSL、Docker、Termux）：  
  ```bash
  hermes gateway
  ```

  Linux/macOS 可安装为后台服务：  
  ```bash
  hermes gateway install # 安装 systemd/launchd 服务
  ermes gateway start # 启动服务
  hermes gateway status # 查看状态
  ```

  测试机器人  
  在 消息平台 中找到你的机器人，发送一条消息：  
  ```text
  You: 你好
  Bot: 你好！我是 Hermes，你的 AI 助手。
  ```

  查看 Gateway 状态  
  ```bash
  hermes gateway status
  ```

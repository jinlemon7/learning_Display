# 聚合函数(COUNT, SUM, AVG, MAX, MIN)
  聚合函数能将‌‌‌多行数据聚合为单个结果，⁢⁢⁢是数据分析和生成报表的重要‏‏‏工具。  
  <img width="2212" height="842" alt="image" src="https://github.com/user-attachments/assets/336428b8-aaed-4346-9cb6-e2433e7caec5" />  

##　COUNT() 函数
  COUNT() 函数用于计算行数或非 NULL 值的数量。  
  计算表中的总记录数：  
  ```sql
  SELECT COUNT(*) AS total_sales
  FROM sales;
  ```

## SUM() 函数
  SUM() 函数用于计算数值列的总和。  
  计算所有课程的总销售额：  
  ```sql
  SELECT SUM(price * sold_count) AS total_revenue
  FROM sales;
  ```

## AVG() 函数
  AVG() 函数用于计算数值列的平均值。  
  计算平均销售量：  
  ```sql
  SELECT AVG(sold_count) AS avg_sales
  FROM sales;
  ```

## MAX() 和 MIN() 函数
  MAX() 和 MIN() 函数分别用于找出数值列的最大值和最小值。  
  查找最高和最低销售量：  
  ```sql
  SELECT 
      MAX(sold_count) AS max_sales,
      MIN(sold_count) AS min_sales
  FROM sales;
  ```

  聚合函数常与 GROUP BY 子句结合使用，对数据进行分组统计。需要注意的是，聚合函数会忽略 NULL 值，除非明确指定（如 COUNT(*) ）  

# 字符串处理函数(UPPER, LOWER, LENGTH等)
  字符串处理‌‌‌函数可以操作和⁢转⁢换⁢文本数据，‏在数‏据清‏洗、格‍式化和‍搜索场‍景中非⁡常重⁡要。  
  <img width="2062" height="544" alt="image" src="https://github.com/user-attachments/assets/dab26a4d-a4e2-4141-a707-cda88e1cd652" />  

## UPPER() 和 LOWER() 函数
  这两个函数分别用于将文本转为大写或小写。  
  ```sql
  SELECT 
      title,
      UPPER(title) AS upper_title,
      LOWER(title) AS lower_title
  FROM forum_posts;
  ```

## LENGTH() 函数
  LENGTH() 函数返回字符串的长度（字节数或字符数，取决于数据库系统）。  
  ```sql
  SELECT title, LENGTH(title) AS title_length
  FROM forum_posts;
  ```

## CONCAT() 函数
  CONCAT() 函数用于连接多个字符串。  
  ```sql
  SELECT 
      CONCAT(author, ': ', title) AS post_info
  FROM forum_posts;
  ```

## SUBSTRING() 函数
  SUBSTRING() 函数可以提取字符串的一部分。  
  ```sql
  SELECT 
      title,
      SUBSTRING(title, 1, 5) AS title_preview
  FROM forum_posts;
  ```

## REPLACE() 函数
  REPLACE() 函数用于替换字符串中的子串。  
  ```sql
  SELECT 
      title,
      REPLACE(title, 'SQL', 'Structured Query Language') AS expanded_title
  FROM forum_posts;
  ```

## TRIM() 函数
  TRIM() 函数用于移除字符串开头和结尾的空格或其他字符。  
  `SELECT TRIM('   编程导航   ') AS trimmed_string;`  

## 时间日期函数(DATE, DATETIME, TIME等)
  常见的时间日‌‌‌期函数包括获取当前时间、⁢⁢⁢提取日期部分、计算日期差‏‏‏异等。不同数据库系统的时‍‍‍间日期函数可能有所不同，⁡⁡⁡这里以 MySQL 为例。

  获取当前日期和时间:  
  ```sql
  SELECT
      NOW() AS `current_datetime`,
      CURDATE() AS `current_date`,
  ```

  | --- | --- | --- |

## 提取日期部分
  从日期时间中提取年、月、日、小时等部分：  
  ```sql
  SELECT 
      post_date,
      YEAR(post_date) AS year,
      MONTH(post_date) AS month,
      DAY(post_date) AS day
  FROM forum_posts;
  ```

 结果：  
 | post_date | year	| month | day |

## 日期格式化
  FORMAT() 函数可以将日期格式化为指定的字符串格式：  
  ```sql
  SELECT 
      post_date,
      DATE_FORMAT(post_date, '%Y年%m月%d日') AS formatted_date
  FROM forum_posts;
  ```

  结果：  
  | post_date |	formatted_date |  
  | --- | --- |  
  | 2025-05-10 | 2025年05月10日 |  
  | 2025-05-12 | 2025年05月12日 |  
  | 2025-05-15 | 2025年05月15日 |  
  | 2025-05-18 | 2025年05月18日 |  

## 日期计算
  可以对日期‌‌进行‌加减操作，计算未⁢⁢来或过去的日期：  
  ```sql
  SELECT 
      post_date,
      DATE_ADD(post_date, INTERVAL 7 DAY) AS next_week,
      DATE_SUB(post_date, INTERVAL 1 MONTH) AS last_month
  FROM forum_posts;
  ```

  结果：  
  | post_date |	next_week	| last_month |  
  | --- | --- | --- |  
  | 2025-05-10 | 2025-05-17 | 2025-04-10 |  
  | 2025-05-12 | 2025-05-19	| 2025-04-12 |  
  | 2025-05-15 | 2025-05-22	| 2025-04-15 |  
  | 2025-05-18 | 2025-05-25	| 2025-04-18 |  

## 日期差异计算
  DATEDIFF() 函数可以计算两个日期之间的天数差异：  
  ```sql
  SELECT 
      post_date,
      CURDATE() AS today,
      DATEDIFF(CURDATE(), post_date) AS days_since_posted
  FROM forum_posts;
  ```

  当前日期 2025-07-30：  
  | post_date |	today	| days_since_posted |  
  | --- | --- | --- |  
  | 2025-05-10 | 2025-07-30	| 81 |  
  | 2025-05-12 | 2025-07-30	| 79 |  
  | 2025-05-15 | 2025-07-30	| 76 |  
  | 2025-05-18 | 2025-07-30	| 73 |  

## 时间戳转换
  可以在 UNIX 时间戳和日期时间之间进行转换：  
  ```sql
  SELECT 
      UNIX_TIMESTAMP('2025-07-30 14:30:00') AS timestamp_value,
      FROM_UNIXTIME(1753081800) AS datetime_value;
  ```

  结果：  
  | timestamp_value	| datetime_value |  
  | --- | --- |  
  | 1753081800 | 2025-07-30 14:30:00 |  

## 实际应用
  时间日期函数在实际应用中十分常见，例如：  
  按月份或季度生成销售报表：  
  ```sql
  SELECT 
      DATE_FORMAT(sale_date, '%Y-%m') AS month,
      SUM(price * sold_count) AS monthly_revenue
  FROM sales
  GROUP BY month
  ORDER BY month;
  ```

  查找最近 30 天内的活跃帖子：  
  ```sql
  SELECT title, author, post_date
  FROM forum_posts
  WHERE DATEDIFF(CURDATE(), post_date) <= 30
  ORDER BY post_date DESC;
  ```

  计算每个帖子的发布时长：  
  ```sql
  SELECT 
      title, 
      post_date,
      CONCAT(
          FLOOR(DATEDIFF(CURDATE(), post_date) / 365), '年 ',
          FLOOR((DATEDIFF(CURDATE(), post_date) % 365) / 30), '月 ',
          DATEDIFF(CURDATE(), post_date) % 30, '天'
      ) AS age
  FROM forum_posts;
  ```

# 练习题
## 练习 1：聚合函数应用
  假设有一个 user_activities 表，记录用户在网站的活动，包含 user_id, activity_type, points_earned（每次活动获得的积分）, activity_date 等字段。编写 SQL 查询，计算每个用户的总积分、平均每次活动获得的积分以及最高单次获得的积分。  

  答案：  
  ```sql
  SELECT 
      user_id, 
      SUM(points_earned) AS total_points,
      AVG(points_earned) AS avg_points,
      MAX(points_earned) AS max_points
  FROM user_activities
  GROUP BY user_id;
  ```

## 字符串函数应用
  假设 users 表中有 email 字段，包含用户的电子邮箱地址。编写一个查询，从邮箱地址中提取用户名部分（@符号之前的部分），并将其转换为全小写。  

  ```sql
  SELECT 
      email,
      LOWER(SUBSTRING(email, 1, INSTR(email, '@') - 1)) AS username
  FROM users;
  ```

## 日期函数应用
  网站上有一个 interviews 表，记录了面试经历分享，包含 id, company, position, interview_date, experience 等字段。编写一个查询，找出最近一个季度（最近 3 个月）内分享的面试经历，并按照面试日期降序排列，同时显示距今多少天。  

  答案：  
  ```sql
  SELECT 
      company,
      position,
      interview_date,
      DATEDIFF(CURDATE(), interview_date) AS days_ago
  FROM interviews
  WHERE interview_date >= DATE_SUB(CURDATE(), INTERVAL 3 MONTH)
  ORDER BY interview_date DESC;
  ```

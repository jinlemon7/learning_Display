# 关于 GitHub Codespaces 的安全日志
  当你在个人帐户拥有的存储库中执行与 GitHub Codespaces 相关的操作时，你可以查看安全日志中的操作。  
  有关访问日志的信息，请参阅 [审查您的安全日志](https://docs.github.com/zh/authentication/keeping-your-account-and-data-secure/reviewing-your-security-log#accessing-your-security-log)。  
  <img width="1440" height="610" alt="image" src="https://github.com/user-attachments/assets/9ca35dcc-1fed-4069-a5d9-32ef16945e34" />

  安全日志包括发生了什么操作以及何时执行的操作等详细信息。

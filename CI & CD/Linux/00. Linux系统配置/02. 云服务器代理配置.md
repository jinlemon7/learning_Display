# 使用 GitHub Container Registry (GHCR) 镜像
  v2rayA 官方除了 Docker Hub，也向 GitHub Container Registry (GHCR) 发布镜像。这通常是一个更稳定、下载速度更快的选择。  
  你可以直接用下面这行命令来拉取： 
  ```bash
  docker pull ghcr.io/v2raya/v2raya:latest
  ```

# 配置 docker-compose.yml
  ```bash
  mkdir -p v2raya
  nano docker-compose.yml
  ```

  复制下面的内容到docker-compose.yml：  
  ```bash
  services:
    v2raya:
      image: ghcr.io/v2raya/v2raya:latest
      container_name: v2raya
      restart: always
      privileged: true
      network_mode: host
      environment:
        - V2RAYA_ADDRESS=0.0.0.0:2017
      volumes:
        - /etc/v2raya:/etc/v2raya
  ```

# 启动 v2rayA 容器
  在 docker-compose.yml 所在目录执行：  
  ```bash
  docker compose up -d
  ```

  v2rayA 图形界面默认端口是2017，别忘了去云服务商防火墙放行  

# 配置 v2rayA
  启动完成后，你可以通过 http://<你的服务器IP>:2017 访问 v2rayA 的 Web 管理界面进行后续配置。  

  1. 创建一个管理员账号

  账号信息位于本地，我们不会上传任何信息到服务器。  
  如不慎忘记密码，使用 v2raya --reset-password 重置。

# 设置网络代理
  设置网络代理，与外部windows一致：  
  ```bash
  sudo nano /etc/environment
  ```

  ```bash
  http_proxy="http://127.0.0.1:10808"
  https_proxy="http://127.0.0.1:10808"
  
  # 如果代理软件支持 SOCKS5
  all_proxy="socks5://127.0.0.1:10808"
  
  # 设置不需要走代理的环境或地址
  no_proxy="localhost,127.0.0.1"
  ```

# 全局软件源设置为阿里云镜像

## Ubuntu / Debian (APT 包管理器)
  对于 Debian 系的系统，主要通过修改 /etc/apt/sources.list 来完成。  
  ```bash
  # 1. 备份原始源文件（安全第一）
  sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
  
  # 2. 用 sed 命令一键替换所有源地址为阿里云镜像地址
  #    注意：${UBUNTU_CODENAME} 需要替换为你的Ubuntu版本代号，如 jammy（22.04）、noble（24.04）
  sudo sed -i 's|http://archive.ubuntu.com|http://mirrors.aliyun.com|g' /etc/apt/sources.list
  sudo sed -i 's|http://security.ubuntu.com|http://mirrors.aliyun.com|g' /etc/apt/sources.list
  
  # 3. 更新软件包列表，使新源生效
  sudo apt update
  ```

## 验证与更新
  配置完成后，最关键的一步是刷新本地软件包列表，以确保系统与新的镜像源建立成功连接。  
  - Debian/Ubuntu 使用 apt：sudo apt update
  - CentOS/RHEL 使用 yum：sudo yum makecache
  - Fedora 使用 dnf：sudo dnf makecache
  - Arch Linux 使用 pacman：sudo pacman -Syy
  - openSUSE 使用 zypper：sudo zypper ref

  检查命令的输出，应该能看到软件包的下载链接是 mirrors.aliyun.com。之后，运行 sudo apt upgrade、sudo yum upgrade 等更新命令时，其下载速度将会有明显提升，就说明镜像源配置大功告成了。

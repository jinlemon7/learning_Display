- 尾递归：从顶到底（“递”），到达最后一层递归层时即可return整个递归函数：递归体放在return子句
- 非尾递归：从顶到底（“递”），再从底到顶（“归”：回溯）。

```java
f(..., int count){    // 非尾递归
  if(count > 20) return ;    // 递归层数超过20，一般会堆栈溢出
  // (1)终止条件

  f(...);    // (2)递归体
  return ;    // (3)返回
}
```
- (1)(2)(3)是非尾递归的三要素

1. 尾递归需要变量保存并传递中间结果到下层递归中
2. 配合Map，简化中间过程的计算————记忆化搜索

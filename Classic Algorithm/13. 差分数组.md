# 定义
  d[i] =  
  - a[0]     i=0
  - a[i]-a[i-1]    i>=1

  性质1：从左到右累加 d[i]，可得到原数组 a   
  性质2：以下两个操作等价  
  - 把 a[i], a[i+1], ..., a[j] 都加 x
  - d[i] + x, d[j+1] - x

  PS：根据性质2，可以把规模区间同种变动在 O(1) 时间内完成  

# 初阶化diff
  ```java
  /**
   * 根据原数组 a 计算差分数组 d
   * 定义：
   *   d[0] = a[0]
   *   d[i] = a[i] - a[i-1]   (i >= 1)
   * 
   * @param a 原数组
   * @return 差分数组 d，长度与 a 相同
   */
  public static int[] computeDiff(int[] a) {
      if (a == null || a.length == 0) {
          return new int[0];
      }
      int n = a.length;
      int[] d = new int[n];
      d[0] = a[0];
      for (int i = 1; i < n; i++) {
          d[i] = a[i] - a[i - 1];
      }
      return d;
  }
  ```

# 更新(区间变更)
  ```java
   /**
   * 区间更新：将原数组 [left, right] 范围内每个元素增加 v
   * @param left  区间左边界（包含）
   * @param right 区间右边界（包含）
   * @param v     增加的值
   */
  public void add(int left, int right, int v) {
      if (left > right || left < 0 || right >= n) return;
      diff[left] += v;
      diff[right + 1] -= v;
  }
  ```

# 还原成原数组
  ```java
  /**
     * 根据差分数组还原原数组
     * @return 还原后的原数组（长度为 n）
     */
  public int[] recover() {
      int[] original = new int[n];
      int prefix = 0;
      for (int i = 0; i < n; i++) {
          prefix += diff[i];
          original[i] = prefix;
      }
      return original;
  }
  ```

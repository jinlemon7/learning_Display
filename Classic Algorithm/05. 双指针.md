# 移动零
  移动零: 将数组中的零全部移动到数组末尾。  
  数组n1:  
  | 0 | 2 | 0 | 3 | 1 | 0 |  
  | --- | --- | --- | --- | --- | --- |  

  虚拟表思想：把原表当空表用  
  ```java
  int j=0;
  for(int i=0; i<n1.length; i++){
    if(n1[i] != 0){
      n1[j++] = n1[i];
    }
  }

  for(j; j<n1.length; j++){
    n1[j] = 0;
  }
  ```

# 环形链表
  <img width="783" height="682" alt="f4586f39dfedc9caf5be1a8e3a42468f" src="https://github.com/user-attachments/assets/1a542dd7-ab6d-4330-b54b-a890b1404ed4" />  

  求环形起点的节点（节点号）C：  
  1. 快指针2步/次，慢指针1步/次
  2. 第1次相遇时，慢指针回到head，快、慢指针都1步/次
  3. 两指针第2次相遇的节点，即环形起点节点C

  数学原理：  
  1. 第1次相遇时，慢指针路程s，快指针路程f  
     f = s + nb = 2s  n: 绕环圈数  
     解得：s = nb, f = 2nb  ，
     此时快慢指针、快指针在环内部的某个节点相遇。因为nb，2nb都是环路程的整数倍，但是有一部分路程是环之外的a  
  3. 第2次相遇时（快慢指针速度均为1）  
     s = a    a: 从开始到圈起点的距离  
     f = a + 2nb    2nb回到环入口  
     此时慢指针、快指针都在环入口。因为此时的2nb都是环路程

## 使用Set解决环形链表
  使用Set添加指针，若有重复，则有环且为环的起点  
  ```java
  Set<ListNode> set = new HashSet<>();
  while (head != null) {
      if (!set.contains(head)) {
          set.add(head);
      } else {
          return head;   // 找到重复节点（环的入口）
      }
      head = head.next;
  }
  return null;  // 无重复，无环
  ```

# 相交链表（双指针）
  <img width="816" height="584" alt="78f789f289a74db46ab8831c9c9b45f6" src="https://github.com/user-attachments/assets/71f500ce-4166-450a-9037-47ce2e507b92" />  

  求相交链表起点的节点（节点号）C：  
  1. p1, p2 遍历head1, head2
  2. p1 = NULL, p2 = NULL时，p1 = head2, p2 = head1; 类似两个链表成环，继续遍历
  3. p1 = p2时，得到节点C或NULL

  数学原理：  
  - p1 = p2 且 != NULL时，a + d + b = b + d + a   
    路程相等，即得节点C  
  - p1 = p2 = NULL时，a + d + b + d = b + d + a + d  
    路程相等，得NULL  

## 使用Set解决相交链表
  遍历head1， 使用Set添加指针  
  遍历head2， 若指针存在于Set，则为相交起点

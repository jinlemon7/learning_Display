# 移动零
  移动零: 将数组中的零全部移动到数组末尾。  
  数组n1:  
  | 0 | 2 | 0 | 3 | 1 | 0 |  
  | --- | --- | --- | --- | --- | --- |  

  虚拟表思想：把原表当空表用  
  ```java
  int j=0;
  for(int i=0; i<n1.length; i++){
    if(n1[i] != 0){
      n1[j++] = n1[i];
    }
  }

  for(j; j<n1.length; j++){
    n1[j] = 0;
  }
  ```

  求环形起点的节点/节点号C：  
  1. 快指针2步/次，慢指针1步/次
  2. 第1次相遇时，慢指针回到head，快、慢指针都1步/次
  3. 两指针第2次相遇的节点，即环形起点节点C

  数学原理：  
  1. 第1次相遇时，慢指针路程s，快指针路程f  
     f = s + nb = 2s  n: 绕环圈数  
     解得：s = nb, f = 2nb  ，
     此时快慢指针、快指针在环内部的某个节点相遇。因为nb，2nb都是环路程的整数倍，但是有一部分路程是环之外的a  
  3. 第2次相遇时（快慢指针速度均为1）  
     s = a    a: 从开始到圈起点的距离  
     f = a + 2nb    2nb回到环入口  
     此时慢指针、快指针都在环入口。因为此时的2nb都是环路程

$x^2 = n$, 求 $x = \sqrt{n}$ = ?  
数学原理：  
$x^2 = n$  
$6 \times 2 = 12$  
$x = \frac{6 + 2}{2} = 4$  
4比2, 6更接近 $\sqrt{12}$  
不断循环：$`\text{res} = \frac{x + \frac{n}{x}}{2}`$, 返回res  

```java
int newton(int n) {
    if (n == 0)
        return 0;
    return (int) sqrt(n, n);
}

double sqrt(double x, int n) {
    double res = (x + n / x) / 2;
    if (res == x)
        return res;
    else
        return sqrt(res, n);
}
```

# Scanner（最常用，适合算法刷题，数据量小 (< 10^5) )
  ```java
  import java.util.Scanner;
  
  public class Main {
      public static void main(String[] args) {
          Scanner sc = new Scanner(System.in);
          
          // 读取整数
          int n = sc.nextInt();
          
          // 读取字符串（遇到空格或换行结束）
          String s = sc.next();
          
          // 读取一整行
          sc.nextLine(); // 注意：如果上一行用了nextInt()，需要先消耗换行符
          String line = sc.nextLine();
          
          // 循环读取直到EOF（常见于未知行数输入）
          while (sc.hasNext()) {
              int x = sc.nextInt();
              // 处理 x
          }
          
          sc.close();
      }
  }
  ```
  - Scanner的构造器可以接受任何类型的输入对象，包括File对象、InputStream、String,或者Readable ( Java 5引入的一个接口.用于描述"具有read()方法的东西”

## 数据读取
  | 方法名 | 简要描述 |  
  | :--- | :--- |  
  | `next()` | 读取并返回下一个完整的标记（默认以空白字符分隔） |  
  | `nextLine()` | 读取并返回当前行的剩余部分，直到遇到换行符 |  
  | `nextByte()` | 读取下一个标记并解析为 `byte` |  
  | `nextShort()` | 读取下一个标记并解析为 `short` |  
  | `nextInt()` | 读取下一个标记并解析为 `int` |  
  | `nextLong()` | 读取下一个标记并解析为 `long` |  
  | `nextFloat()` | 读取下一个标记并解析为 `float` |  
  | `nextDouble()` | 读取下一个标记并解析为 `double` |  
  | `nextBoolean()` | 读取下一个标记并解析为 `boolean` |  
  | `nextBigInteger()` | 读取下一个标记并解析为 `BigInteger` |  
  | `nextBigDecimal()` | 读取下一个标记并解析为 `BigDecimal` |

# BufferedReader（高效，适合大数据量(> 10^6) ）
  ```java
  import java.io.*;
  
  public class Main {
      public static void main(String[] args) throws IOException {
          BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
          
          // 读取一个整数
          int n = Integer.parseInt(br.readLine());
          
          // 读取一行并分割成字符串数组
          String[] parts = br.readLine().split(" ");
          int a = Integer.parseInt(parts[0]);
          int b = Integer.parseInt(parts[1]);
          
          // 循环读取直到EOF
          String line;
          while ((line = br.readLine()) != null) {
              // 处理每行
          }
          
          br.close();
      }
  }
  ```
  - 比 Scanner 快 3-5 倍，数据量 10^6 以上建议使用

  | 方法名 | 简要描述 |  
  | :--- | :--- |  
  | `read()` | 读取单个字符，返回字符的整数编码（0-65535），到达流末尾返回 -1 |  
  | `read(char[] cbuf, int off, int len)` | 读取字符到字符数组的指定位置，返回实际读取的字符数，到达流末尾返回 -1 |  
  | `readLine()` | 读取一行文本，以换行符 `\n`、回车符 `\r` 或回车换行符 `\r\n` 为结束标志，返回不包含行终止符的字符串，到达流末尾返回 null |  
  | `lines()` | 返回一个 `Stream<String>`，其元素为从该 BufferedReader 中读取的行（Java 8+） |  
  | `skip(long n)` | 跳过指定数量的字符，返回实际跳过的字符数 |  
  | `ready()` | 检查流是否已准备好被读取（缓冲区非空或底层流已就绪） |  
  | `markSupported()` | 检查此流是否支持 `mark()` 操作，BufferedReader 始终返回 `true` |  
  | `mark(int readAheadLimit)` | 标记流中的当前位置，后续调用 `reset()` 可重新定位到此标记点 |  
  | `reset()` | 将流重置到最近一次标记的位置 |  
  | `close()` | 关闭流并释放所有关联的系统资源 |
